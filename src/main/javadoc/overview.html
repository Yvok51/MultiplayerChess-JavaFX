Multiplayer Chess is a multiplayer game of chess in which two players join a match and play until the match ends or one of them resigns.

The server is started by calling mvn exec:java in the root directory. The client application is started with mvn javafx:run.

The server has a fixed port number and as such only one instance can run at a time. The IP address the client connects to is also fixed as localhost. The port number of the server and the IP address of the client is set in the multiplayerchess/multiplayerchess/common/networking/Networking.java file.

The server and client communicate by sending messages over TCP using Java serialization. The SocketMessageListener and SocketMessageWriter are used to send and receive messages. Both work in their own threads. The messages are written using a queue that the SocketMessageWriter reads from. The SocketMessageListener has a callback function that is called when a message is received. This callback function then performs routing of the message to the correct handlers.
